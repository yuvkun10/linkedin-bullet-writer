# LinkedIn Bullet Writer

A local React app for turning rough bullet points into polished LinkedIn post drafts.

## Features

- Bullet-to-post draft generation with deterministic local logic
- Tone controls for confident, warm, bold, and practical writing
- Hook styles for insight, story, question, and contrarian openings
- Short, standard, and expanded draft variants
- Hook, body, CTA, and hashtag composition
- Copy-to-clipboard and text export actions

## Development

```bash
npm ci
npm run dev
```

## Verification

npm run lint

npm run build
